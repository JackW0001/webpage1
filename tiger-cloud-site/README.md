# Tiger Trade × The Cloud Choir Responsive Landing Page

This is a static responsive website built from the poster structure. It uses local HTML, CSS, JavaScript, and cropped image assets from the supplied reference poster.

## How to use

1. Open `index.html` in a browser.
2. Replace the images in `assets/` with final high-resolution brand/event images if needed.
3. Update the button links inside `index.html` for Auckland Live, Ticketmaster, and Tiger Trade campaign pages.

## Mobile support

The layout uses CSS grid, `clamp()` font sizing, and media queries for tablet and phone screens. On mobile, the navigation becomes a menu, cards stack vertically, and large desktop image rows are simplified.
