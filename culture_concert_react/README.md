# Culture Concert React Website

React + Vite version of the uploaded poster-style website.

## Structure

```text
culture_concert_react/
├── index.html
├── package.json
├── public/
│   └── assets/
│       ├── poster.png
│       ├── logo-strip.png
│       ├── footer-logo-strip.png
│       ├── hero-stage.png
│       ├── globe-card.png
│       ├── choir-card.png
│       └── icon crops from the uploaded poster
└── src/
    ├── main.jsx
    ├── App.jsx
    └── App.css
```

All image assets are taken from the uploaded poster image. No external/generated image assets are used.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```
