# Culture Concert Website

A responsive single-page website based on the supplied poster reference.

## How to run

Open `index.html` directly in your browser.

For a local server, you can also run:

```bash
python -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## Files

- `index.html` — page structure
- `style.css` — visual style and responsive layout
- `script.js` — mobile menu open/close
- `assets/` — cropped image assets from the reference poster
